Backend Chrome Extension
========================

Enables quick access to the backyard screen, 'chrome://' and 'about:' pages.

[Chrome Web Store](https://chrome.google.com/webstore/detail/eiingmhoibigiplopcmepmahoihedjnk)


ABILITY
-------
- chrome://extensions
- chrome://downloads
- chrome://history
- chrome://bookmarks
- about:plugins
- about:cache
- about:memory
- about:stats
- about:dns
- about:network
- about:crash
- about:hang
- about:internets
- about:histograms
- about:version
- Themes gallery page
